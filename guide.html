User Guide

1. Buka WEB

Akses aplikasi melalui URL yang diberikan atau buka file HTML di browser yang mendukung.

2. Izin Kamera

Berikan izin untuk mengakses kamera saat diminta oleh browser.

3. Arahkan Kamera

Arahkan kamera perangkat Anda ke target gambar yang telah ditentukan (contoh: gambar target dari MindAR).

4. Interaksi Model 3D

Setelah target gambar dikenali, model 3D akan muncul di atas target. Anda bisa melihat model dari berbagai sudut dengan menggerakkan perangkat.
